# E-Commerce React Application

A frontend-focused E-Commerce web application built using React.js.  
This project demonstrates real-world React concepts such as Context API, custom hooks, protected routes, and reusable components.

Live Demo: https://e-commerce-pearl-zeta-98.vercel.app/

---

## Features

- Product listing
- Add to cart and remove from cart
- Update product quantity
- Cart total calculation
- Login and Register pages (frontend)
- Protected routes for authenticated users
- Global state management using Context API
- Custom hooks for reusable logic
- Centralized API handling using Axios

## Project Structure

src/
├── api/                # Axios configuration and endpoints
├── Components/         # Reusable UI components
├── Context/            # Auth and Cart contexts
├── hooks/              # Custom hooks
├── Pages/              # Page-level components
├── utils/              # Helper and validation functions
├── App.jsx             # Root component
└── index.js            # Entry point

## Installation and Setup

1. Clone the repository

git clone https://github.com/Abiabi01/E-Commerce.git

2. Navigate to the project folder

cd E-Commerce/ecommerce

3. Install dependencies

npm install

4. Start the development server

npm start

The application will run on http://localhost:3000
